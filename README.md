Weather App
A simple weather application built with React and TypeScript. Users can search for a city and view its current temperature and wind speed using the Open-Meteo API.

Features
Search weather by city
Display current temperature
Display wind speed
Loading and error handling
Responsive and clean UI
Built with React + TypeScript
Technologies
React
TypeScript
CSS
Open-Meteo API
